# دليل استخدام وإدارة بوت تليجرام المدعوم بنماذج الذكاء الاصطناعي

## مقدمة

هذا الدليل يشرح كيفية استخدام وإدارة بوت تليجرام المدعوم بنماذج الذكاء الاصطناعي. البوت يوفر الوصول إلى مجموعة متنوعة من نماذج الذكاء الاصطناعي مثل ChatGPT، GPT-4، Claude، Gemini، وغيرها، بالإضافة إلى خدمات إنشاء الصور والفيديو والموسيقى.

## متطلبات التشغيل

- Python 3.8 أو أحدث
- حساب تليجرام
- رمز بوت تليجرام (Token) من BotFather
- مفاتيح API لنماذج الذكاء الاصطناعي المختلفة

## تثبيت وإعداد البوت

### الخطوة 1: تنزيل الكود

```bash
git clone https://github.com/NaJiKuN/AIPoweredProBot.git
cd AIPoweredProBot
```

### الخطوة 2: تثبيت المتطلبات البرمجية

```bash
pip install -r requirements.txt
```

### الخطوة 3: إعداد ملف البيئة

قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع وأضف المتغيرات التالية:

```
TOKEN=رمز_البوت_الخاص_بك
ADMIN_ID=معرف_المسؤول_الرئيسي
```

### الخطوة 4: إعداد قاعدة البيانات

```bash
python database_setup.py
```

### الخطوة 5: تشغيل البوت

```bash
python bot.py
```

## هيكل المشروع

البوت مقسم إلى وحدات منفصلة لتسهيل الصيانة والتطوير:

- `bot.py`: الملف الرئيسي للبوت
- `config.py`: ملف التكوين والإعدادات
- `database_setup.py`: إعداد قاعدة البيانات
- `modules/`: مجلد يحتوي على وحدات البوت المختلفة
  - `users.py`: إدارة المستخدمين
  - `admins.py`: إدارة المسؤولين
  - `api_keys.py`: إدارة مفاتيح API
  - `subscriptions.py`: إدارة الاشتراكات
  - `models.py`: إدارة نماذج الذكاء الاصطناعي
  - `notifications.py`: إدارة الإشعارات
  - `stats.py`: إدارة الإحصائيات
  - `commands.py`: إدارة الأوامر العامة

## أوامر المستخدم

### الأوامر الأساسية

- `/start`: بدء استخدام البوت والحصول على معلومات عامة
- `/help`: عرض قائمة الأوامر والمساعدة
- `/account`: عرض معلومات الحساب والاشتراك
- `/premium`: عرض خيارات الاشتراك المميز
- `/settings`: إعدادات البوت ونماذج الذكاء الاصطناعي
- `/privacy`: عرض شروط الاستخدام وسياسة الخصوصية

### أوامر النماذج والمحتوى

- `/model`: اختيار نموذج الذكاء الاصطناعي المفضل
- `/deletecontext`: حذف سياق المحادثة
- `/s`: البحث على الإنترنت باستخدام Perplexity
- `/photo`: إنشاء الصور
- `/video`: إنشاء الفيديو
- `/midjourney`: استخدام Midjourney لإنشاء الصور
- `/suno`: إنشاء الأغاني

### قواعد الاشتراكات والحزم

- لا يمكن شراء نفس الحزمة أكثر من مرة في الشهر

- 1 طلب = 1 إدخال سواء كتابة أو صورة أو فيديو أو صوت
- جميع الأسعار يتم التعامل معها بعملة ⭐ النجوم الخاصة بتليجرام

## أوامر المسؤول

- `/admin`: عرض قائمة أوامر المسؤول
- `/add_admin [user_id]`: إضافة مسؤول جديد
- `/remove_admin [user_id]`: إزالة مسؤول
- `/list_admins`: عرض قائمة المسؤولين
- `/add_api [api_name] [api_key]`: إضافة مفتاح API جديد
- `/remove_api [api_name]`: إزالة مفتاح API
- `/list_apis`: عرض قائمة مفاتيح API
- `/broadcast [message]`: إرسال رسالة لجميع المستخدمين
- `/stats`: عرض إحصائيات البوت
- `/toggle_package [package_name]`: تفعيل/تعطيل حزمة
- `/list_packages`: عرض قائمة الحزم المتاحة

## إدارة مفاتيح API

### إضافة مفتاح API جديد

```
/add_api [api_name] [api_key]
```

مثال:
```
/add_api gpt-4 sk-abc123...
```

### إزالة مفتاح API

```
/remove_api [api_name]
```

مثال:
```
/remove_api gpt-4
```

### عرض قائمة مفاتيح API

```
/list_apis
```

## استكشاف الأخطاء وإصلاحها

### مشكلة: البوت لا يستجيب

1. تأكد من أن البوت قيد التشغيل
2. تحقق من صحة رمز البوت في ملف `.env`
3. تأكد من اتصال الإنترنت

### مشكلة: خطأ في مفاتيح API

1. تحقق من صحة مفتاح API المضاف
2. تأكد من اختيار النموذج الصحيح
3. تحقق من توفر رصيد كافٍ في حساب API

### مشكلة: عدم ظهور بعض النماذج

النماذج تظهر فقط عند توفر مفاتيح API الخاصة بها. تأكد من إضافة مفاتيح API للنماذج التي ترغب في ظهورها.

### مشكلة: خطأ في قاعدة البيانات

1. تأكد من تشغيل `database_setup.py` قبل تشغيل البوت
2. تحقق من صلاحيات الكتابة في المجلد

### مشكلة: عدم تلقي النجوم

1. تأكد من ربط البوت بـ bot method api لتلقي النجوم
2. تحقق من إعدادات الدفع في لوحة تحكم BotFather
